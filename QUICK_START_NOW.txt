HOW TO USE FEATURES
════════════════════════════════════════════════════════════════════════════

TO SEARCH:
  1. Look at search box (already has Zop.dev keywords)
  2. Change keywords to search for different topics
  3. Results update automatically
  4. Shows top 10 posts ranked by relevance

TO FILTER:
  1. Click "⚙️ Advanced Filters" to expand
  2. Adjust slider: minimum relevance score
  3. Select dropdown: time window (24h, 3d, 7d, 30d)
  4. Check boxes: which subreddits to include
  5. Filters apply instantly
  6. Your filters persist when you refresh

TO REFRESH POSTS:
  1. Click "🔄 Refresh" button
  2. Dashboard reloads latest posts from Reddit
  3. Your filter settings stay active
  4. Page loads in 1-5 seconds

TO SAVE POSTS:
  1. Find a post you like in Feed tab
  2. Click "✓ Engage" button on that post
  3. See confirmation message
  4. Post moves to "Saved Posts" tab
  5. Data saves automatically

TO VIEW ON REDDIT:
  1. Find a post in Feed or Saved Posts
  2. Click "🔗 Open Link" (or "View" in Saved Posts)
  3. Post opens in new Reddit tab
  4. Read full discussion, comments, etc.

TO MANAGE SAVED:
  1. Click "Saved Posts" tab
  2. See all your saved posts
  3. Review relevance scores
  4. Click "✖ Unsave" to remove any
  5. Your list updates immediately

TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════

If dashboard doesn't load:
  → Make sure URL is exactly: http://localhost:8501
  → Check if the server is running (should say it's running)
  → Try refreshing the page (F5 or Cmd+R)

If posts don't appear:
  → Click "🔄 Refresh" button to reload
  → Check your filters - you might be filtering out all posts
  → Try changing keywords in search box

If "Engage" button doesn't work:
  → Wait 1 second and try again
  → Check your internet connection
  → Try refreshing the page

If saved posts disappeared:
  → Go to "Saved Posts" tab
  → Data should be there
  → If missing, try refreshing page (F5)
