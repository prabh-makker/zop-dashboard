HOW TO USE YOUR DASHBOARD
════════════════════════════════════════════════════════════════════════════════

1. OPEN DASHBOARD
   → Go to: http://localhost:8501
   → You should see the clean white interface

2. BROWSE FEED (Main Tab)
   → Shows latest posts with Zop.dev keywords
   → Default: "FinOps OR AWS cost OR cloud optimization OR infrastructure"
   → Top 10 most relevant posts displayed

3. SEARCH POSTS
   → Search box is pre-filled with Zop.dev keywords
   → Change keywords to search different topics
   → Results update automatically
   → Top 10 ranked by relevance

4. USE FILTERS
   → Click "⚙️ Advanced Filters" to expand
   → Adjust minimum relevance (0-100%)
   → Select time window (24h, 3d, 7d, 30d)
   → Choose subreddits
   → Filters apply instantly and PERSIST on refresh

5. REFRESH POSTS
   → Click "🔄 Refresh" button anytime
   → Reloads latest posts from Reddit
   → Your filter settings stay active

6. SAVE POSTS
   → Find a post you like
   → Click "✓ Engage" button
   → Post moves to "Saved Posts" tab
   → Data saves automatically

7. VIEW ON REDDIT
   → Click "🔗 Open Link" on any post
   → Opens Reddit post in new tab

8. MANAGE SAVED POSTS
   → Click "Saved Posts" tab
   → See all your saved posts
   → Click "✖ Unsave" to remove any

TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════════

Dashboard won't load:
  → Make sure URL is: http://localhost:8501 (exactly)
  → Try refreshing browser (Ctrl+R or Cmd+R)
  → Check if server is running

Posts don't appear:
  → Click Refresh button
  → Check if filters are too restrictive
  → Try changing search keywords

Buttons don't work:
  → Wait 1 second and try again
  → Check internet connection
  → Try refreshing page

Data disappeared:
  → Go to "Saved Posts" tab
  → Data should be there
  → If missing, try page refresh
