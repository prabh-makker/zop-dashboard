HOW TO USE
================================================================================

1. OPEN DASHBOARD
   - Get URL from Streamlit Cloud
   - Open in browser
   - Wait for page to load (shows "Fetching Reddit posts...")

2. BROWSE POSTS
   - See 100-300 posts from Reddit
   - Click on post title to read
   - Engage with filters below

3. FILTER POSTS
   - Time: Select 24h, 3d, 7d, or 30d
   - Subreddit: Check which ones to show
   - Relevance: Slide to set minimum score
   - Search: Type keywords to find

4. SAVE POSTS
   - Click engagement button on any post
   - Post moves to "Saved" section
   - View saved posts anytime

5. REFRESH
   - Click "Refresh Data" to get fresh posts
   - Useful if no posts showing or need latest
